## Why This Round

## Changes In This Version

1. 引入 `RACI-lite` matrix：request / own / execute / review / approve。
2. 大任务默认必须出现 owner、executor、reviewer 三个槽位。
3. manager 不默认拥有执行权。

## Locked Principle

组织化的核心不是更多人，而是职责不重叠。

## Residual Risk
