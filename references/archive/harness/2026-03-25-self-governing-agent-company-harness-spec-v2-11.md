## Why This Round

“像公司一样组织”不能停留在 metaphor；必须把角色族和责任边界写成合同。

## Changes In This Version

1. 把 agent 组织固定成 `principal / manager / lead / worker / reviewer` 五类角色族。
2. 每类角色只允许承担有限职责集合。
3. 禁止 role title 与 write authority 脱钩。

## Locked Principle

角色不是称号，是权限与责任合同。

## Residual Risk

1. 还没定义角色与路径 ownership 的映射
2. 还没定义 manager 与 reviewer 是否可兼任
