This is a compatibility entrypoint for the [research bundle](../research/SKILL.md).

Read [../research/manifest.toml](../research/manifest.toml).
Read [../research/SKILL.md](../research/SKILL.md).

Operate in `memo` mode and create a research memo that separates:

- evidence
- counter-evidence
- unknowns
- recommendation

Prefer [../research/templates/research-memo.md](../research/templates/research-memo.md).
Read [../research/refs/volatile-research-default.md](../research/refs/volatile-research-default.md).
Read [../research/refs/internal-research-routing.md](../research/refs/internal-research-routing.md).
Read [../research/refs/task-artifact-routing.md](../research/refs/task-artifact-routing.md).
For volatile topics, create or reference a research dispatch, then gather fresh external sources or source-note references before recommending.
If a work item is in play, write to `.harness/tasks/<task-id>/attachments/`.
Only when `advanced governance mode` is explicitly enabled and the memo is truly cross-task, promote it to a governance workspace location.
