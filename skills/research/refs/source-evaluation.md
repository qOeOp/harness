# Source Evaluation

Every durable source should be judged on four axes:

1. freshness
   - how likely the fact has changed
2. authority
   - official, primary, high-signal secondary, or weak
3. relevance
   - how directly it answers the scoped question
4. support strength
   - whether it clearly supports a concrete claim

## Practical rubric

1. `official`
   - vendor docs, repo source, standards body, regulator, first-party API
2. `primary`
   - original paper, filing, benchmark repo, first-hand announcement
3. `high-signal-secondary`
   - strong technical analysis or industry review with verifiable citations
4. `weak`
   - SEO summary, repost, uncited forum aggregation

If a weak source contributes a useful lead, convert the lead into a stronger source before relying on it.
